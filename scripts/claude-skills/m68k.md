# SKILL: m68k — Motorola 68000 pour Mega Drive

## Registres
```
D0-D7   data registers (32-bit)
A0-A6   address registers (32-bit)
A7      stack pointer (SP)
PC      program counter
SR      status register: T.S..III...XNZVC
        X=extend N=negative Z=zero V=overflow C=carry
```

## Tailles opérandes : .b (byte) .w (word=16) .l (long=32)
Défaut = .w si omis. Registres Dn/An toujours 32-bit en interne.

## Instructions fréquentes
```asm
move.l  D0, D1          ; copie
moveq   #imm, Dn        ; move quick: imm -128..127, extension signe 32b, 1 cycle
movem.l D0-D7/A0-A6, -(SP)  ; push registres
movem.l (SP)+, D0-D7/A0-A6  ; pop registres
lea     addr, An        ; load effective address
pea     addr            ; push effective address

add/sub/and/or/eor.l Dn, Dn
addi/subi/andi/ori/eori.l #imm, Dn
addq/subq #1..8, Dn     ; rapide (1 cycle)
mulu    Dn, Dn          ; multiply unsigned (16x16→32)
muls    Dn, Dn          ; multiply signed
divu    Dn, Dn          ; divide unsigned (32/16→quot16+rem16 en Dn)

lsl/lsr/asl/asr/rol/ror.l #n, Dn  ; shift/rotate
swap    Dn              ; échange mots haut/bas de Dn
ext.l   Dn              ; extension signe .w→.l

bra/bsr/bcc label       ; branch (relative, court si ±128, long sinon)
jmp/jsr addr            ; jump absolu
rts                     ; return from subroutine
rte                     ; return from exception (restaure SR+PC)
nop

tst.l   Dn              ; set flags, no result
cmp.l   Dn, Dn          ; set flags (Dn-Dn)
dbcc    Dn, label       ; decrement branch: Dn--, branch si !cond && Dn!=-1
```

## Modes d'adressage
```
Dn / An                 ; register direct
(An)                    ; register indirect
(An)+                   ; post-increment
-(An)                   ; pre-decrement
d(An)                   ; displacement (16-bit)
d(An, Dn.l)             ; indexed
$AAAA.w / $AAAAAAAA.l   ; absolute (16-bit sign-extended ou 32-bit)
#imm                    ; immediate
```

## Cycles utiles (68000 NTSC = 7.67 MHz)
```
moveq           4 cycles
move.l Dn,Dn    4 cycles
add.l  Dn,Dn    8 cycles
mulu            (54+2n) cycles — n = nb bits à 1 dans la source
divu            ~140 cycles max — LENT, éviter en gameplay
dbf             10 cycles (branch) / 14 (no branch = fin boucle)
jsr/rts         16/16 cycles
```

## Patterns typiques Mega Drive

### Boucle rapide sur N éléments
```asm
    moveq   #N-1, D7       ; dbf compte de N-1 à -1
.loop:
    ; ... traitement ...
    dbf     D7, .loop
```

### Lire/écrire VDP data port (DMA-safe)
```asm
    move.l  #$40000000, VDP_CTRL   ; set write address VRAM $0000
    move.w  D0, VDP_DATA           ; write word
```

### Vecteurs d'exception (offset dans la table $000000)
```
$000   Reset SP initial
$004   Reset PC
$008   Bus Error
$00C   Address Error
$010   Illegal Instruction     ← breakpoint logiciel (ILLEGAL = $4AFC)
$014   Divide by Zero
$018   CHK
$01C   TRAPV
$020   Privilege Violation
$028..  Trace, Line-A, Line-F
$060   Spurious Interrupt
$064   Level 1 (external)
$068   Level 2 (VDP H-Blank)  ← HInt
$06C   Level 3
$070   Level 4 (VDP V-Blank)  ← VInt
$080..  TRAP #0..#15
```

## Pièges communs
- `move.b` vers An non supporté — utiliser `move.w` + `ext.l`
- Address Error si accès .w/.l sur adresse impaire
- Bus Error si accès hors espace mémoire valide
- `mulu`/`muls` utilisent seulement les 16 bits bas des deux opérandes
- SR accessible en mode superviseur seulement (`move SR, Dn` → `move CCR, Dn` en user mode)
- Retour d'interruption : `rte` (restaure SR), pas `rts`

## GDB m68k utile
```
(gdb) info registers        # D0-D7, A0-A7, PC, SR
(gdb) x/10wx $a7            # stack dump (SP)
(gdb) x/i $pc               # instruction courante
(gdb) set $d0 = 0x42        # modifier registre
(gdb) x/40bx 0xffff0000     # dump RAM
```
